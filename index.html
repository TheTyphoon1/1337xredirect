1337x Redirector

This page is designed to work as a redirector for BitTorrent search plugins. If you use it normally (without a query), you will see these instructions.

How to use with Windows BitTorrent 7.11

  1. Open your BitTorrent UI Extras search engine page.
  2. Add a new line with the following format:
    3. x1337x|https://thetyphoon1.github.io/1337xredirect/?q=%s
  3. Now, when you search in BitTorrent, you can type without having to format spaces.
  4. The redirector will automatically convert spaces to + and redirect to https://x1337x.cc.

Notes

  • This redirect only happens when a query is provided (e.g., ?q=your search).
  • If you visit this page normally, it will display these instructions.
